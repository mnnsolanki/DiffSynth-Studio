## Quick Command Reference

```bash
# PHASE 1: Extract Embeddings (5 min)
python examples/wanvideo/prepare_face_embeddings.py \
    --face_images_dir <YOUR_FACES> \
    --output_dir <OUTPUT>

# PHASE 2: Train LoRA (2-10 hours)
python examples/wanvideo/model_training/train_face_lora.py \
    --face_images_dir <YOUR_FACES> \
    --dance_videos_dir <DANCES> \
    --pose_videos_dir <POSES> \
    --model_path <WAN_MODEL> \
    --output_dir <OUTPUT> \
    --num_epochs 100

# PHASE 3: Generate Video (2-5 min)
python examples/wanvideo/face_identity_inference.py \
    --face_images_dir <YOUR_FACES> \
    --face_lora_path <TRAINED_LORA> \
    --pose_video <POSE_VIDEO> \
    --output <OUTPUT_VIDEO>
```
